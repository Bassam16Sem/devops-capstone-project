# Proyek Pengembangan Aplikasi Web

## Deskripsi Proyek
Proyek ini adalah aplikasi web sederhana yang dirancang untuk mendemonstrasikan integrasi CI/CD menggunakan GitHub Actions. Setiap kali ada perubahan kode yang diunggah (push), sistem akan secara otomatis menjalankan proses build untuk memastikan tidak ada kesalahan dalam kode.

## Fitur Utama
- Integrasi otomatis dengan GitHub Actions.
- Lencana status build real-time.
- Dokumentasi terstruktur.

## Cara Penggunaan
1. Clone repositori ini.
2. Jalankan `npm install` untuk menginstal dependensi.
3. Jalankan `npm test` untuk memverifikasi fungsionalitas.
